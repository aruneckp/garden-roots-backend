# Garden Roots Backend - AI Assistant Context

## Project Overview
FastAPI-based REST API for mango e-commerce platform with Oracle database. Supports multiple user roles (customer/admin/delivery) and payment processing.

## Key Technologies
- **Framework**: FastAPI with Python 3.x
- **Database**: Oracle DB (local XE + cloud ADB support)
- **ORM**: SQLAlchemy with connection pooling
- **Auth**: Google OAuth 2.0 + JWT
- **Payment**: HitPay (primary), Stripe (secondary)

```python
# Dependency for database sessions
def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()
```
